# 推理资讯周报

这是一个基于 **GitHub Pages** 的静态周报页面，用来展示每周精选的推理书单。

## 使用方式

1. 将本仓库部署到 GitHub Pages。
2. 确保 `data/weekly.json` 已由自动任务更新。
3. 打开首页即可查看本周精选、评分和豆瓣详情链接。

## 说明

- 页面数据来自豆瓣移动端接口生成的周报文件。
- 封面图优先使用远程图片，失败时会自动显示内嵌占位图。
- 精选规则使用固定评分阈值，便于稳定展示。

## 本地预览

如果你想本地查看，可以用静态服务器打开目录，例如：

- `python -m http.server`
- `npx serve`

然后访问 `index.html`。
